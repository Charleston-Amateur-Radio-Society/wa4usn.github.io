N1MM Logger+ is a free, full-featured contest logging program designed for amateur radio operators. It's become the standard for many contesters due to its powerful features and active development.

## Getting Started

To get started with N1MM+, download the latest version from the official website. The installation is straightforward and includes all the necessary components.

## Key Features

- Real-time scoring and rate displays
- Integrated bandmap with DX cluster support
- Support for virtually all contests
- CW, SSB, and digital mode support
- Networked logging for multi-operator stations

## Resources

Check out the N1MM website for comprehensive documentation and user guides. Our club also has several experienced N1MM users who can help you get set up.
